Local Recall for People Who Refuse Surveillance

Mnema is a Tauri + Rust desktop agent that continuously captures your screen, embeds it with CLIP, and stores every trace in an on-device LanceDB. Search is semantic, privacy is absolute, and the interface is built for scripting your own rebellion.

Why Mnema Exists

We reject captive recall products that siphon data to corporate clouds. Mnema is the antidote: offline, scriptable, and aligned with The Authentic Rebellion Framework's demand for human dignity.

Privacy

Everything stays local

No telemetry, no network calls, no API keys. Screenshots, vectors, OCR text, and hashes never leave your hardware.

Agency

Hackable overlay

Tauri frontend exposes events and commands so you can wire automations—launch playlists when VS Code appears, fire prompts when a meeting starts, or export a daily digest.

Human Flourishing

Memory with consent

Mnema honors human context: blurred faces by default, cached OCR for quick redaction, and file-version fingerprints so collaboration remains accountable.

Architecture at a Glance

Purpose-built for DirectML acceleration and human-readable scripting.

  • Rust backend (Tauri) orchestrates capture, ONNX sessions, LanceDB writes, and new OCR/face-blur commands.
  • Frontend is plain HTML/CSS/JS so contributors can fork the UI or embed custom widgets.
  • TrOCR + YuNet models run locally; SentencePiece tokenizer ships with the app for offline decoding.
  • Thread-safe caches ensure OCR responses are instant after the first pass.

Privacy & Safety

Default Blur Pipeline

YuNet face detection (DirectML) runs during capture, saving blurred thumbnails. Originals remain on disk and unlock only when requested.

Immutable File Fingerprints

BLAKE3 hashes—full or segmented—link a memory to the precise file version open at capture time.

Scriptable Consent

Because the UI is bare HTML/JS, you can inject automation that pauses capture, redacts regions, or syncs with your own rituals.
